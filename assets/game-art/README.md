# Game art sources

## Streak flame

- Artwork: **Pixel Fire** by DaneIsAlive
- Source: https://opengameart.org/content/pixel-fire
- Original file: `fire.png` (six 24×24 frames in a 144×24 sprite sheet)
- Local file: `streak-fire.png`
- License: [CC0 1.0](https://creativecommons.org/publicdomain/zero/1.0/)
- Downloaded: 2026-06-28

The source page identifies this as hand-created pixel art for a game jam. It is
used for streak and fire-related UI in place of platform-dependent emoji.
